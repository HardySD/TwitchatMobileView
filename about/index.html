About Twitchat Mobile View

How does it work?

Basically, this extension use JavaScript to translate the chat's node below Video Player. Also use CSS to apply position absolute and z-index to required elements (e.g. viewer counter, stream time, ...)

It detects when the window size changes and if the window width is less than or equal to 919px, then the Mobile View is activated whereas if the window width is greater than 919px Twitch is restored to normal (Normal View).

Why 919px?

Because when Twitch detects that the window is less than or equal to 919px then it hides the chat.

What if the url changes?

The extension has a background script that detects url changes and if the chat exists after changing the url.

If the chat exists then it checks the window width to enable the Mobile View or not.

if the chat does not exist then it does nothing.
