About Twitchat Mobile View

This extension only works for Twitch: https://www.twitch.tv where it detects the stream chat.

How does it work?

Basically, this extension detects when the window size changes and if the window width is less than or equal to 919px, then the Mobile View is activated whereas if the window width is greater than 919px Normal View is restored.

Why 919px?

Because when Twitch detects that the window width is less than or equal to 919px then it hides the chat.

Twitchat Mobile View solves this problem for those who want to chat at any time.
